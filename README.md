# Machine Learning Based Smart Gloves for Sign Language Translation

## Overview
This project presents a Machine Learning and Deep Learning based Smart Gloves System for real-time sign language translation. The system is designed to help deaf and mute individuals communicate easily with people who do not understand sign language.

The smart glove uses flex sensors and an accelerometer to capture finger bending and hand movements. The collected sensor data is processed using Machine Learning algorithms to recognize gestures and convert them into text and speech output.

## Working Principle
1. Flex sensors detect finger bending.
2. Accelerometer captures hand orientation and movement.
3. Sensor data is sent to the microcontroller.
4. Machine Learning algorithms classify the gesture.
5. Recognized gesture is converted into text.
6. Text is converted into speech using TTS.
